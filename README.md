# 바닐라 자바스크립트 기반의 공적 마스크 판매처 검색 애플리케이션(front-end)

### 설명

- 현재 사용자의 위치를 기준으로 검색한 주변 마스크 판매처를 지도에 표시하는 기능

### 개발환경

- 공적 마스크 판매 현황 조회 API
  https://app.swaggerhub.com/apis-docs/Promptech/public-mask-info/20200307-oas3#/
- Kakao 지도 API
  http://apis.map.kakao.com/web/guide/
- vanilla.js
- Express.js(npm)
- HTML5, CSS3
